Страница Bot-a на Aiogram3 (bot.html)

Расположен в домашней папке "/home/ $PROJECT_USER / $PROJECT_NAME /html_for_bot"

Для доступа Nginx к вашей домашней папке /home/... , необходимо пользователя www-data добавить в группу своего пользователя проекта (marat):

$ sudo usermod -aG marat www-data

Файл конфигурации /etc/nginx/sites-available/nginx_bot.conf
